# Text List

## Panoramica
Componente che permette di creare un elenco flessibile con due tipi di righe:
- **Righe titolo**: mostrano solo un testo (senza link)
- **Righe link**: mostrano un link cliccabile

Ogni elemento dell'elenco può essere configurato come titolo o come link in modo indipendente.

## Come utilizzare

### Configurazione di ogni elemento

Per ogni elemento dell'elenco, puoi scegliere tra due modalità:

#### Modalità Titolo/Testo
1. Attiva il campo **Mostra Testo o Link**
2. Compila il campo **Testo Elenco** con il testo che vuoi visualizzare (massimo 25 caratteri)

#### Modalità Link
1. Lascia disattivato il campo **Mostra Testo o Link**
2. Compila il campo **Testo Link** con il testo del link (massimo 45 caratteri) - *obbligatorio*
3. Compila il campo **Link URL** con l'indirizzo di destinazione - *obbligatorio*
   - Puoi inserire un URL diretto oppure selezionare un elemento link esistente

## Note importanti
- Ogni elemento dell'elenco è indipendente: puoi avere alcuni elementi come titoli e altri come link nello stesso elenco
- I campi obbligatori devono essere compilati per poter salvare l'elemento
- Se mancano informazioni necessarie, l'elemento potrebbe non essere visualizzato correttamente
